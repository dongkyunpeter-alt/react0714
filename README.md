#### 깃으로 관리하기
다음 명령어들은 Git으로 프로젝트를 관리할 때 자주 사용하는 기본 흐름입니다.

1. `git init`: 현재 폴더를 Git 저장소로 초기화합니다.
2. `git add .`: 변경된 파일을 Git에 등록할 준비(스테이지) 상태로 만듭니다.
3. `git commit -m "first commit"`: "first commit"이라는 메시지로 현재 변경 내용을 저장합니다.
4. `git branch -M main`: 현재 브랜치 이름을 main으로 변경합니다.
5. `git remote add origin https://github.com/dongkyunpeter-alt/react-test.git`: GitHub 원격 저장소를 연결합니다.
6. `git push -u origin main`: 로컬의 main 브랜치를 GitHub에 업로드합니다.
